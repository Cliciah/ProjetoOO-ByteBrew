## ☕ Sobre o Projeto
Este repositório é destinado à implementação do trabalho prático da disciplina de Orientação por Objetos. O objetivo é modelar e desenvolver um sistema de gerenciamento de vendas e fidelidade para a cafeteria temática "Byte & Brew", utilizando a linguagem Java e aplicando os pilares da programação orientada a objetos.
